关于本项目

这是一个交互式现生蜥形纲演化可视化项目，按“鸟类到目，其余到科”的粒度展示龟鳖、喙头蜥、鳄类、鸟类与有鳞类的系统关系。

数据与技术说明

  • 数据基准：分类系统主要参考 Reptile Database 与 IOC World Bird List；层级上对节点过密的支系补入了总科、下目等中间层。
  • 鸟类主干：鸟类末级采用 IOC 当前目级框架；Neoaves 内部主干优先参照近年广泛使用的系统发育结果，避免沿用已被并回的旧目级拆分。
  • 时间标尺：高阶节点优先参考文献常见的冠群分化时间；末级节点时间为便于可视化整理的近似冠群时间。
  • 图像生成：当前阶段优先完成树结构与数据整理，图像资源仍待后续按节点逐步补充。
  • 技术架构：本项目基于 D3.js (2D) 与 Three.js (3D) 混合开发。为了提升国内访问体验，所有图像资源均采用 CDN 链路进行加速分发。

操作指南

  • 拖拽画布：自由移动演化树视图
  • 滚轮/双指：缩放查看演化细节
  • 点击节点：展开或收起演化分支
  • 点击文字：查看物种/科目的详细资料卡片
